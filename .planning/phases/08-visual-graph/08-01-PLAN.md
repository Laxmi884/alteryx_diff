**Function 1: `build_digraph`**

Signature:
```python
def build_digraph(
    result: DiffResult,
    all_connections: tuple[AlteryxConnection, ...],
    all_nodes: tuple[AlteryxNode, ...],
) -> nx.DiGraph:
```

Logic:
- Build sets: `added_ids` from `result.added_nodes`, `removed_ids` from `result.removed_nodes`, `modified_ids` from `result.modified_nodes` (use `nd.tool_id`), `conn_changed_ids` from `result.edge_diffs` (union of src_tool and dst_tool)
- Priority order: added > removed > modified > connection > unchanged
- For each node in `all_nodes`, assign status using priority. Call `int(node.tool_id)` for the integer node id.
- `G.add_node(tool_id, label=node.tool_type, color=COLOR_MAP[status], status=status, title=f"{node.tool_type} | {status}")`
- For each connection in `all_connections`: `G.add_edge(int(conn.src_tool), int(conn.dst_tool))`
- Return G

Note: `all_nodes` should be the UNION of old and new workflow nodes. GraphRenderer (Plan 02) is responsible for passing the correct combined node set. build_digraph is purely a mapper.

**Function 2: `hierarchical_positions`**

Signature:
```python
def hierarchical_positions(G: nx.DiGraph) -> dict[int, tuple[float, float]]:
```

Logic:
- Handle cycles: `dag = G.copy()`, loop while not `nx.is_directed_acyclic_graph(dag)`, find cycle with `nx.find_cycle(dag)`, remove `cycle[-1][:2]` back-edge
- Assign layers: `for layer, nodes in enumerate(nx.topological_generations(dag)): for node in nodes: dag.nodes[node]["layer"] = layer`
- Layout: `raw_pos = nx.multipartite_layout(dag, subset_key="layer", align="vertical")`
- raw_pos values are numpy arrays — convert: `{int(node): (float(coords[0]) * LAYOUT_SCALE, float(coords[1]) * LAYOUT_SCALE) for node, coords in raw_pos.items()}`
- Return positions dict

**Function 3: `canvas_positions`**

Signature:
```python
def canvas_positions(
    nodes_old: tuple[AlteryxNode, ...],
    nodes_new: tuple[AlteryxNode, ...],
) -> dict[int, tuple[float, float]]:
```

Logic:
- Populate `pos` dict with old node positions first, then override with new (covers added/modified)
- `for node in nodes_old: pos[int(node.tool_id)] = (node.x, node.y)`
- `for node in nodes_new: pos[int(node.tool_id)] = (node.x, node.y)`
- Return pos
