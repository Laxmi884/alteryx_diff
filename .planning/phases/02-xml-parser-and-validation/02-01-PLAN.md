Public API (locked in CONTEXT.md):
  def parse(path_a: pathlib.Path, path_b: pathlib.Path) -> tuple[WorkflowDoc, WorkflowDoc]
  Calls _parse_one(path_a) first, then _parse_one(path_b). Fails on first error — no multi-error collection.

Internal helpers (Claude's discretion per CONTEXT.md):

_parse_one(path: pathlib.Path) -> WorkflowDoc
  Stage 1 — pre-flight:
    if not path.exists(): raise MissingFileError(filepath=str(path), message=f"File not found: {path}")
    if not path.is_file(): raise UnreadableFileError(filepath=str(path), message=f"Path is not a regular file: {path}")
  Stage 2 — parse:
    parser = etree.XMLParser(recover=False)
    try: tree = etree.parse(str(path), parser)  # str() required — NOT pathlib.Path directly
    except etree.XMLSyntaxError as exc: raise MalformedXMLError(filepath=str(path), message=f"Malformed XML in {path.name}: {exc}") from exc
    except OSError as exc: raise UnreadableFileError(filepath=str(path), message=f"Cannot read {path}: {exc}") from exc
  Stage 3 — convert:
    return _tree_to_workflow(tree, filepath=str(path))

_tree_to_workflow(tree: etree._ElementTree, filepath: str) -> WorkflowDoc
  root = tree.getroot()
  Extract nodes: root.findall("Nodes/Node") — safe even if Nodes element is absent
  For each node_elem:
    tool_id = ToolID(int(node_elem.get("ToolID")))
    gui = node_elem.find("GuiSettings")
    plugin = gui.get("Plugin", "") if gui is not None else ""
    pos = gui.find("Position") if gui is not None else None
    x = float(pos.get("x", "0")) if pos is not None else 0.0
    y = float(pos.get("y", "0")) if pos is not None else 0.0
    config_elem = node_elem.find("Properties/Configuration")
    config = _element_to_dict(config_elem) if config_elem is not None else {}
    Append AlteryxNode(tool_id=tool_id, tool_type=plugin, x=x, y=y, config=config)
  Extract connections: root.findall("Connections/Connection") — safe if Connections absent
  For each conn_elem:
    origin = conn_elem.find("Origin")
    dest = conn_elem.find("Destination")
    if origin is None or dest is None: continue  # skip malformed connection silently
    Append AlteryxConnection(src_tool=ToolID(int(origin.get("ToolID"))), src_anchor=AnchorName(origin.get("Connection", "Output")), dst_tool=ToolID(int(dest.get("ToolID"))), dst_anchor=AnchorName(dest.get("Connection", "Input")))
  Return WorkflowDoc(filepath=filepath, nodes=tuple(nodes_list), connections=tuple(connections_list))

_element_to_dict(elem: etree._Element) -> dict[str, Any]
  Recursive helper. Convention (chosen to serve Phase 3 normalizer and Phase 5 differ):
    - XML attributes become "@key" entries: for k, v in elem.attrib.items(): result[f"@{k}"] = v
    - Non-whitespace text becomes "#text": if elem.text and elem.text.strip(): result["#text"] = elem.text.strip()
    - Child elements keyed by tag name; repeated same-tag siblings promoted to list automatically:
        if tag already in children_by_tag and existing is not a list: children_by_tag[tag] = [existing, child_dict]
        elif tag already in children_by_tag and existing is a list: existing.append(child_dict)
        else: children_by_tag[tag] = child_dict
  result.update(children_by_tag); return result

Type annotations: all functions must carry full type annotations. lxml internal types use from lxml import etree then etree._ElementTree and etree._Element. mypy --strict may not resolve lxml private types fully; use "etree._ElementTree[etree._Element]" or cast as needed to satisfy strict mode without using Any for the tree parameter.
